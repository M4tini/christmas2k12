# Christmas2k12
First HTML5 canvas, featuring:

* sine scroller with bitmap font
* canvas gradients
* ascii nfo
* mouse tracking snow

Preview: <https://rawgit.com/M4tini/christmas2k12/master/index.html>

### Resources
* 8bit Chiptune Christmas Medley, by Fantomenk
* Chunky ST/Amiga style font, by matt0
